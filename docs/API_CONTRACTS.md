# API Contracts

## Node API

### `GET /api/health`

Returns gateway health and persistence mode.

### `GET /api/sessions/recent`

Returns the latest stored searches from MongoDB or the in-memory fallback store.

### `POST /api/search`

Request:

```json
{
  "query": "I need to build a resume but I have no design skills",
  "limit": 8
}
```

Response:

```json
{
  "query": "I need to build a resume but I have no design skills",
  "summary": "For \"I need to build a resume but I have no design skills,\" start with Canva, LinkedIn, and Google Docs so you can move from exploration into your first practical step.",
  "results": [],
  "agentTrace": [],
  "meta": {
    "cache": "miss",
    "persistenceMode": "memory"
  }
}
```

### `GET /api/guides/:slug.pdf?query=...`

Streams a one-page starter PDF generated by the FastAPI service.

## FastAPI Service

### `GET /health`

Returns service health and knowledgebase size.

### `POST /search`

Receives the user query and returns ranked recommendations plus the agent trace.

### `GET /guides/{slug}.pdf?query=...`

Builds a single-page PDF starter guide for the requested tool.
